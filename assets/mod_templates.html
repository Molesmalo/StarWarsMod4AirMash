Attention! If you use Requestly to load StarMash, this update requires a new configuration file: that you can find here!.
I you haven't already updated it yet, please follow this steps:

  • 1- Open this link and then right click on the page, on the context menu select "Save as..." to download the new configuration file to your device.
  • 2- Open Requestly's configuration page, by clicking the blue 'r' icon next to the address bar.
  • 3- Select all the existing rules (by clicking the box to the left of "Name & Description").
  • 4- Click the trash bin icon, in the upper right corner of the page.
  • 5- Confirm the deletion. The rules list should now be empty.
  • 6- Click the "Upload Rules" blue button in the upper right corner of the page.
  • 7- Select the file you downloaded in step 1.
  • 8- Close Requestly's configuration page.
  • 9- Go back to AirMash and hit: Ctrl+F5 you are on Windows, or Cmnd+Shift+R on Mac to force a full refresh.
  • 10- Enjoy StarMash!
